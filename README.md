# AshaSathi AI

> AI Workflow Agent for Smarter Frontline Healthcare

## Vision

AshaSathi AI is an Android-based AI workflow agent designed to assist ASHA workers by reducing paperwork, identifying high-risk patients, recommending next actions, and automating healthcare workflows.

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Firebase
- Gemini API
- Google AI Studio
- Room Database

## Status

🚧 Currently under active development for the Google AI Agent Builder Series 2026.
