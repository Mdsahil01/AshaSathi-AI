# AshaSathi AI

> AI Workflow Agent for Smarter Frontline Healthcare

## 🚀 About

AshaSathi AI is an Android application that helps ASHA workers reduce administrative work by using AI to assist with patient visits, risk identification, and follow-up planning.

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- Firebase (Planned)
- Gemini AI (Planned)
